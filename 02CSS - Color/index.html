Input Types

  • Normal Predefined color names
    •             h1.{
                color:Red;
            }
  • RGB - Red Green Blue Values each ranges from 0 - 256
    •             h1.{
                color:rgb(58, 125, 4);
            }
  • RGBA - Red Green Blue Values with Alpha. Alpha is for tranperancy ranges from 0.0 to 1.0
    •             h1.{
                color:rgb(58, 125, 4,0.75);
            }
  • Hexadecimal Values - has 6 digit
    •             h1.{
                color:#240275;
            }
  • HSL - Hue (0 degree to 360 deg)- color wheel, Saturation (0% - 100%) - color shade, Lightness(0% -100%) - greyness level
    •             h1.{
                color:hsl(40,100%,50%);
            }
  • HSLA - with Alpha
    •             h1.{
                color:hsl(40,100%,50%,0.5);
            }
