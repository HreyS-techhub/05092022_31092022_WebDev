CSS

CSS stands for Cascading Style Sheets

SYNTAX

        p{
            Property:Value;
        }

3 TYPES OF STYLING

  • Inline Styling - Written within the specific Tag
  • Internal Styling - Written within the html page inside the Head Tag using the style tag
  • External Styling - We link an external style sheet(.css) inside the head tag using link tag

    • Command in css

    /* command statement */

    Selectors in CSS

  • Element selector - Accessed by using Tag
    •             p{
                Property:Value;
            }
  • ID Selector - Accessed by using ID Name leading "#"
    •             #idName{
                Property:Value;
            }
  • Class Selector - Accessed by using Class Name leading "."
    •             #idName{
                Property:Value;
            }
